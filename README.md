# Simple Neural Network Digit Recognizer

A pure NumPy implementation of a neural network for MNIST digit recognition, featuring backpropagation, multiple activation functions, and weight persistence.

## Features

- **Pure NumPy**: No external deep learning frameworks required
- **Flexible Architecture**: Configurable layer sizes and activation functions
- **Optimization Options**: SGD and Momentum optimizers
- **Weight Persistence**: Save/load trained models
- **Early Stopping**: Automatic training termination when performance plateaus
- **MNIST Support**: Built-in MNIST dataset loader
